▶️ How to Run

Week 1 — First API Endpoint

1. Use the existing workspace: If you are already working in this folder, you can skip cloning and use the existing workspace directly.

2. Navigate to the week-1 folder:

cd assignments/week-1
3. Install dependencies:

npm install
4. Start the server:

node api-endpoint.js
5. Test the endpoint:

Open Postman or your browser and send a request to:

GET http://localhost:3000/
GET http://localhost:3000/time
